<게임설명>
-오목은 상대방보다 먼저 대각선, 가로, 세로 방향 중 하나로 5개의 연속된 돌을 놓으면 승리하는 게임이다. 
-4목일 경우 4개의 연속된 돌을 놓으면 승리.

<게임방법>
1. 저장된 게임을 불러오려면 y를 누르고 파일이름을 입력한다.
2. 새 게임을 하려면 n을 누르고 오목판의 세로길이와 가로길이를 설정해준 후 플레이어 수를 입력해준다. (이때 플레이어 수가 2명이면 오목, 3명이면 4목이 진행된다.)
3. 방향키로 (오목판 내에서만) 커서를 이동시킬 수 있다.
4. 각 사용자가 돌을 번갈아 가며 둔다.(Enter키 또는 SPACE키 사용 가능) 
     ※돌이 있는 위치에는 돌을 둘 수 없다. 
  -player1,2,3의 돌은 각각 'O','X','Y'이다. 
5. 상대방보다 먼저 대각선, 가로, 세로 방향 중 하나로 5개의 연속된 돌을 놓으면 승리한다. (4목일 경우 4개의 연속된 돌을 놓으면 승리.)
6. 게임 도중 1을 누르면 중간에 파일저장이 가능하다.
7. 게임 도중 2를 누르면 게임을 저장하지 않고 나갈 수 있다.
8. 누군가 한 명이 이길 경우 1,2 키는 적용되지 않으며 아무 키나 '두 번' 누르면 빠져나간다. 

<부록>
-UNIX기반 library 'ncurses.h'를 사용하여 구현하였다. 'ncurses.h'는 유닉스 환경 상에서 커서를 움직이거나 색깔을 추가하는 등의 기능을 사용하고 싶을 때 이용하는 library로 이번 프로젝트에서는 ncurses로 구현한 화면상(WINDOW *win)에 while 문을 사용하여 저장된 오목판을 계속적으로 출력하는 형식을 사용하였다.

- int echo() int noecho()
//사용자에 의해 타이핑된 입력을 echo하기위해 echo()를 지정하고 침묵하기 위해서는 noecho()를 지정한다.
- int keypad(win, bf)
//TRUE이면, 입력을 위해 기다리고 있을 때 사용자 단말기의 키보드 상에서 키패드의 사용이 가능하다. Ncurses는 키패드의 기능(function)과 화살표(arrow) 키들을 위해 KEY_* 처럼 .h에 정의된 키 코드를 반환할 것이다. 이것은 숫자 블록(numerical block)과 커서 키들을 사용할 수 있으므로 PC 키보드에서 매우 유용하다.
- refresh(); wrefresh(win); 
//화면을 갱신
- int printw(fmt, ...) int wprintw(win, fmt, ...) int mvprintw(y, x, fmt, ...)
//printf와 역할이 비슷함

등의 ncurses.h에서 사용가능한 함수들을 이용하였다.
